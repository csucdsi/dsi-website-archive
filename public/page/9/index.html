Installing R with Homebrew on Mac OS X

By Justin Bankes on March 24, 2017

This document explains how to install R and RStudio with Homebrew on Mac OS. Main Instructions The steps below will install Homebrew, R, and RStudio. Open a terminal and run the following commands. Install Homebrew. usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" This command will download, install, and set up Homebrew. The first time a brew command runs it will ask for a password. Install R with the following commands: brew tap homebrew/science # adds another source for applications.

About us

The Chico State Data Science Initative strives to foster a data-empowered society working within a culture of equity-minded open science. We do so by connecting people to opportunities to learn new skills and engage in impactful research across the spectrum of disciplines.
